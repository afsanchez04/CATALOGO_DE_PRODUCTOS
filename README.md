# 📦 APP_CATALOGO_PRODUCTOS

> Un catálogo interactivo y dinámico diseñado bajo la metodología ágil Scrum para la gestión y visualización de productos.

---

## 📋 Product Backlog (Historias de Usuario)


| Rol | Acción | Beneficio |
| :--- | :--- | :--- |
| **Cliente** | Ver una lista de todos los productos | Conocer la oferta disponible en la plataforma. |
| **Cliente** | Filtrar productos por categorías | Encontrar rápidamente lo que estoy buscando. |
| **Cliente** | Ver detalles específicos de un producto | Conocer las características al hacer clic en él. |
| **Administrador** | Agregar, editar o eliminar productos (CRUD) | Mantener el catálogo actualizado en tiempo real. |
